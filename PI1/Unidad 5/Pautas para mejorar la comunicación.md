
Para que el mensaje cumpla su objetivo, se deben seguir estas recomendaciones:

| Pauta                         | Descripción Clave                                                                                            |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------ |
| **Claridad de propósito**     | Saber exactamente qué se quiere comunicar antes de emitir el mensaje.                                        |
| **Codificación inteligible**  | Evitar tecnicismos innecesarios; usar símbolos familiares para el receptor.                                  |
| **Consulta y Participación**  | No planear en el vacío; pedir opiniones para ajustar el mensaje al entorno.                                  |
| **Foco en el Receptor**       | Considerar las necesidades y beneficios (incluso a largo plazo) de quien recibe la información.              |
| **Tono y Credibilidad**       | Mantener coherencia entre lo que se dice y cómo se dice para evitar dudas.                                   |
| **Realimentación (Feedback)** | La comunicación solo se completa cuando se confirma la comprensión del mensaje.                              |
| **Gestión Emocional**         | Reconocer que la comunicación también trata con sentimientos y motivaciones.                                 |
| **[[Escuchar]]**              | La comunicación efectiva es responsabilidad no sólo del emisor, sino también del receptor de la información. |
